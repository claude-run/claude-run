## 🛠️ How to Create a GitHub Profile README

### Step 1: Create a Repository
1. Go to [GitHub](https://github.com) and sign in
2. Click the **+** icon in the top-right corner
3. Select **New repository**
4. **Important**: Name the repository exactly as your GitHub username
   - For example: If your username is `claude-run`, name it `claude-run`
5. Make sure the repository is **Public**
6. Click **Create repository**

### Step 2: Add README.md
1. In your new repository, click **Create new file**
2. Name the file `README.md` (must be exact)
3. Add your profile content (see example below)
4. Scroll down and click **Commit changes**

### Step 3: Customize Your Profile
Your README.md will automatically appear on your GitHub profile page!
